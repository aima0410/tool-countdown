# カウントダウンタイマー
カウントダウンタイマーのWebミニアプリです。<br />
シンプルで直感的な操作ができます。

## **React + TypeScript + Vite**
ホスティングサービスには、**GitHub Pages** を採用しました。
**GitHub Actions** を使って自動でデプロイしています。
